# MAToken 加密钱包项目

MAToken 是一款面向 B2B 和 B2C 市场的多链加密钱包解决方案。

## 项目结构

- `homepage.html` - **MAToken 官网首页（企业官网）** ⭐
- `index.html` - MAToken 钱包下载页面
- `logo.jpg` - MAToken Logo
- `商业服务方案.md` - **商业服务方案（简洁版）** ⭐
- `B2B_B2C_Business_Plan.md` - B2B/B2C 商业服务方案（详细版）
- `Product_Design_Plan.md` - 产品设计方案

## 快速开始

1. **打开 `homepage.html` 查看官网首页** ⭐ - 包含完整的 B2B/B2C 服务展示
2. 打开 `index.html` 查看钱包下载页面
3. 阅读 `商业服务方案.md` 了解商业模式
4. 阅读 `Product_Design_Plan.md` 了解产品架构与功能
